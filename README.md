# Last session

Quick start in local
---

1. npm i -g saku
1. saku init
1. saku dev

Tasks
---

Configured tasks based on [saku](https://github.com/kt3k/saku).

- **init** : `saku init`
  > initialize for development
- **dev** : `saku dev`
  > start develop server
- **build** : `saku build`
  > build distributions on production
- **fmt** : `saku fmt`
  > format source code

Global dependencies
---

- **node** >= v8.5.0
- **npm** >= v5.5.1
- **saku** >= v0.11.0
